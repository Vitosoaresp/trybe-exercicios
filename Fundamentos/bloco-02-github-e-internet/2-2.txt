Exercicio passado no dia 2
1. Navegue até a raiz do projeto com o arquivo .txt;
    
    cd trybe-exercicios
    
2. Verifique se não existe nada sem *"commitar"* utilizando `git status` :
    
    git status
    
3. Crie uma nova `branch` com o nome `trybe-skills-changes` e faça checkout nela;
git branch trybe-skills-changes
git checkout trybe-skills-changes
4. No arquivo `.txt` , ao final da lista de habilidades , adicione mais duas habilidades que serão desenvolvidas na Trybe:
cat >> trybe_skills.txt
git add trybe_skills.txt
git commit -m "adiciona mais abilidades"
git push -u origin trybe-skills-changes
5. Abra um *Pull Request* com uma descrição detalhada:
6. Retorne para a branch principal, `master` , com o comando: `git checkout master` ;
git checkout master
7. Verifique que você está na branch `master` , com o comando: `git branch` (esta branch deve estar com o formato original, sem as habilidades recém adicionadas);
    
    git branch
    
8. Crie uma nova `branch` `trybe-skills-updates` a partir da `master` e faça checkout nela;
git branch trybe-skills-update
git checkout trybe-skills-update
9. No mesmo arquivo `.txt` que você modificou no *passo 4* , também ao final da sua lista de habilidades, adicione mais um aprendizado que você terá nos próximos blocos:
cat >> trybe_skills.txt
    
    git add trybe_skills.txt
    git commit -m "adiciona mais uma abilidade"
    git push -u origin trybe-skills-update
    
10. Agora, faça o `merge` das *branches* `trybe-skills-changes` e `trybe-skills-updates` na `branch` `master` , através do *Pull Request* :
